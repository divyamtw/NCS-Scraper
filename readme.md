# NCS Song Scraper

A simple Node.js utility to scrape NCS (NoCopyrightSounds) songs from the web.

## Features

- **Auto mode**: Scrapes all available pages automatically.
- **Manual mode**: Specify a page range to scrape.


## Usage

### Auto mode

Runs the scraper over the full range of pages:
```bash
npm start
```

### Manual mode

Specify the starting and ending page numbers:
```bash
npm start <starting page> <ending page>
```

Example:
```bash
npm start 1 10
```
